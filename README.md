# Player Score Calculator (Python)
## Project Description
The **Player Score Calculator** is a simple Python program designed to demonstrate fundamental programming concepts such as **user input handling**, **data type conversion**, **arithmetic operations**, and **formatted output**.
The program calculates the **average score per game** based on user-provided inputs.
##  Objectives
